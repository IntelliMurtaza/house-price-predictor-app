# 🏠 House Price Predictor (End-to-End ML Project)

This project is an end-to-end Machine Learning system that predicts house prices based on property features and provides an interactive web application.

---

## 🚀 Features

- 📊 Data preprocessing using pipelines  
- 🤖 Model comparison (Linear Regression & Random Forest)  
- ⚙️ Hyperparameter tuning  
- 🌐 Interactive web app using Streamlit  
- 💰 Real-time price prediction  
- 📉 Confidence estimation  
- 🌍 Multiple house comparison  

---

## 🧠 Key Learning

This project focuses not only on model building but on creating a complete ML pipeline and deploying it as a usable product.

---

## ⚠️ Future Improvements

- Implement advanced models like XGBoost  
- Improve model performance
